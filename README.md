Motivación del Proyecto

Este software nace de una necesidad real. Junto con mi pareja, llevamos adelante un emprendimiento de venta de indumentaria. Al notar que la gestión del stock se volvía compleja a medida que crecíamos, decidí volcar mis conocimientos como estudiante de programación para digitalizar el proceso.

Lo que empezó como una solución para cargar datos (ID, talle, color y cantidad), tiene una hoja de ruta clara:

Fase 1 (Actual): Interfaz de usuario (UI) completa y funcional con Tkinter.

Fase 2: Persistencia de datos mediante conexión a MySQL para asegurar la integridad de la información.

Fase 3 (Próximamente): Implementación de Dashboards interactivos con Matplotlib. El objetivo es visualizar métricas clave, como cuáles son las prendas más vendidas o el nivel de rotación de talle, permitiéndonos tomar decisiones basadas en datos para nuestro negocio.

Clothing Store CRUD - Sistema de Gestión de Stock
Este es un sistema de gestión de inventario desarrollado en Python utilizando la librería Tkinter para la interfaz gráfica. El proyecto nace de la necesidad de administrar el stock de un emprendimiento textil de forma eficiente y profesional.

Características
Interfaz Gráfica (UI): Diseño limpio y organizado con validaciones visuales.

Gestión de Productos: Permite la carga de prendas con detalles de ID, Nombre, Color, Cantidad y Talle.

Visualización en Tiempo Real: Implementación de un Treeview para visualizar el stock actual de manera tabular.

Operaciones CRUD: Preparado para las funciones de Crear, Leer, Actualizar y Borrar registros.

Tecnologías Utilizadas

Lenguaje: Python 3.x

Librería UI: Tkinter / TTK

Base de Datos: MySQL 

Análisis y Visualización: Matplotlib (Para la futura implementación de dashboards y gráficos estadísticos de ventas y stock)

Control de Versiones: Git & GitHub
